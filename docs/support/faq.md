# Soporte para personas mayores usuarias de ezivia

Este flujo busca ofrecer ayuda inmediata y comprensible sin depender de conexión a internet.
Consta de dos partes: una **FAQ impresa** que acompaña el dispositivo y un **formulario
opcional** para familiares o cuidadores.

## Guía rápida impresa

| Tema | Explicación corta |
|------|-------------------|
| Encender el móvil | Mantén pulsado el botón lateral hasta que aparezca el logotipo de ezivia. |
| Volver a la pantalla principal | Pulsa el botón redondo "Inicio" que siempre se muestra en la parte inferior. |
| Llamar a un contacto favorito | Desde la pantalla principal toca la foto o nombre. La llamada comenzará de inmediato. |
| Enviar alerta SOS | Mantén pulsado el botón rojo durante 3 segundos; suelta si no deseas enviarla. |
| Abrir Ajustes | Si tocas "Ajustes" sin mantener pulsado subir volumen, aparecerá un aviso único. Mantén presionado el botón de subir volumen hasta que vibre y luego toca "Ajustes" una sola vez para entrar. |
| Ajustar el tamaño del texto | Pulsa "Ajustes" → "Tamaño de letra" y elige la opción que sea más cómoda. |
| Silenciar llamadas | Pulsa el botón lateral de volumen y selecciona "Silencio". |

### Consejos de uso seguro
- Carga el teléfono cada noche para que siempre esté disponible.
- Si se muestra un mensaje que no entiendes, pulsa el botón "Ayuda" para obtener asistencia.
- Guarda únicamente contactos de confianza en la lista rápida.

## Formulario opcional de soporte

Para incidencias complejas, familiares o cuidadores pueden rellenar este formulario (digital o
impreso) y entregarlo al equipo de soporte de ezivia.

1. **Datos de contacto del remitente** (nombre y correo/teléfono, se guardan únicamente para
   devolver la llamada).
2. **Persona usuaria afectada**: alias o iniciales (evitar datos completos para mantener la
   privacidad).
3. **Descripción del problema**: explicar qué estaba haciendo, qué esperaba y qué sucedió.
4. **¿El problema ocurre siempre?** Sí/No.
5. **Adjuntar capturas o fotografías** (opcional).
6. **Prioridad percibida**: Alta (no puede usar el móvil), Media, Baja.
7. **Consentimiento informado**: casilla para autorizar el tratamiento de los datos proporcionados
   según la política de privacidad.

El equipo de soporte debe registrar cada envío en la herramienta interna y usar `AnalyticsEvent`
(`SupportRequested`) para medir qué temas generan más consultas sin exponer datos personales.
